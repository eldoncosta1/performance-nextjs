# Performance React

Aplicação para trabalhar os fundamentos de performance no React.js

## Fundamentos

- Renderizações no React
- Exemplo de aplicação
- React DevTools

## Otimizações

- Memo

- Formatação de dados
- Dynamic import (Code Splitting)
- Virtualização
- Bundle Analyzer
